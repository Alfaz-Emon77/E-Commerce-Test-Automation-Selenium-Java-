# E-Commerce Test Automation (Selenium + Java)

## Project Overview
This project demonstrates **automation testing of a demo e-commerce website** using **Selenium WebDriver and Java**.  
It simulates typical user actions including searching products, applying filters, adding to cart, and interacting with products for QA demonstration.

> ⚠️ **Note:** This project is for **educational/demo purposes** only.

---

## Tools & Technologies
- **Selenium WebDriver** – Browser automation  
- **Java** – Programming language  
- **ChromeDriver** – Browser driver  
- **Git/GitHub** – Version control and portfolio showcase  

---

## Automated Tasks / Test Scenarios
1. Open the e-commerce website  
2. Search for a product (e.g., “Laptop” or “Mobile”)  
3. Apply filters (brand, price, category)  
4. Select the first product from search results  
5. Add product to cart  
6. View product details  
7. Sort products (e.g., Price: Low to High)  
8. Navigate back to search results  
9. Close browser
